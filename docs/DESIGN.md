# Design System: 30-repo-demo-v1

## Visual Philosophy
The design follows a **Zenith Alpha / Technical Nexus** aesthetic. It uses a "Deep Space Black" foundation with "Nexus Neon" (#a5f3fc) technical accents, focusing on portfolio culmination, grid complexity, and interactive transparency.

## Design Patterns
-   **Nexus Nodes**: Translucent black containers (Black-40) with 1px white borders (5% opacity) and expressive neon-border hover states.
-   **Nexus Gradient**: Specialized CSS-driven background gradient with noise overlays and high-fidelity radial glows to signify the "Zenith Point."
-   **Category Pills**: High-contrast interactive buttons with neon background fills for active states and monospaced typographical tracking.
-   **Technical Briefings**: Side-aligned technical descriptors with monospaced metadata and Git-orange accents to maintain ecosystem consistency.

## Color Palette
-   **Zenith Space**: Primary Foundation (Black)
-   **Nexus Neon**: Primary Action & Status (#a5f3fc)
-   **Deep Slate**: Secondary Text & Metadata (Slate-500)
-   **Audit Emerald**: Verification Success (Emerald-500)

## Interaction Design
-   **Gallery Entrances**: 1000ms duration translate-scale animations for the hero typographical blocks.
-   **Pill Kinetics**: 300ms color-state transitions for gallery filters to ensure snappy category switching.
-   **Zenith Motion**: Subtle noise pulsation and background orb migrations to create a "Living Portfolio" feel.
